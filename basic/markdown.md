# 文本

使用"\#"加空格创建标题
```
# 标题
```
# 标题

使用“\*\*”加粗文字

```
**加粗**
```
**加粗**

使用center标签居中
```
<center>居中</center>
```
<center>居中</center>

使用“\>”引用一段话
```
 > 引用
 ```
 > 引用

+ 使用“\+”，“\-”或者“\*”加空格创建无序列表
```
+ 列表
```
- 列表


使用"1.","2."创建有序表格
```
1. 苹果
2. 香蕉
```
1. 苹果
2. 香蕉

# 特殊文本
添加链接
```
<http://herui.blog-harryx.cn>
```
```
[somestring](http://somelink.cn)
```

使用3个"`"添加代码
```
print "hello world!"
```
